# How to use

1. cell 실행
 ``
ctrl + E : run cell
shift + E : run cell + select below
alt + E : run cell + insert below


2. cell type

code : y
markdown : m
raw code : r


3. cell 상태

초록 : edit
파랑 : command

edit -> command : esc
command -> edit : enter


4. cell 추가

위에 추가 : a(above)
아래 추가 : b(below)
